## Overview
CrosshairX is a lightweight and powerful custom crosshair overlay tool designed for competitive FPS gamers. It allows you to create, customize, and overlay professional crosshairs with full control over color, size, opacity, outline, and dynamic recoil patterns. The tool works on top of any game including Valorant, CS2, CS:GO, Apex Legends, and more. Perfect for players who want a precise and consistent aim experience with advanced crosshair customization on Windows 11.

## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| App doesn't start | Run as Administrator and disable antivirus temporarily |
| Crosshair not visible | Enable overlay and run in compatibility mode |
| Game overlay issues | Disable fullscreen optimizations |
| Crosshair disappears | Restart CrosshairX and game |
| Performance drop | Lower crosshair complexity and opacity |
| Color not applying | Update to latest version 2026 |
| Windows 11 compatibility error | Update Windows and graphics drivers |
| Download / update failed | Use the latest release and check your internet connection |
